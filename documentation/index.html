#Zoom Meeting

You can manage your zoom meeting.

  1. You can create/edit/delete a zoom meeting only admin users permitted.
  2. New Meeting Create
  3. Upcoming Meeting List
  4. Live Meeting List
  5. Previous Meeting List
  6. Zoom Settings
Generating API Key and Secret

For this zoom meeting you’ll be using JWT token method to make the API connection. Please note, this method is only account level connection.

  1. First goto https://marketplace.zoom.us/develop/create
  2. Click on Develop on top of the page and build app page if you directly did not go into https://marketplace.zoom.us/develop/create page
  3. Click JWT and Create

  4. Click Create
  5. On the next screen you should see Information, App Credentials, Feature and Activation menus
  6. Fill in your basic information about the App.
  7. On the Credentials Page. Copy both API key and API Secret after you have filled all the details in information page.

  4. You can change administrator recipient mail address(any mail address).
